常用注解

@RestController 和 @Controller

  • 相同点
  1. 元注解：@Target({ElementType.TYPE})，作用在类、接口、枚举类上
  2. 元注解：@Retention(RetentionPolicy.RUNTIME)，运行期保留
  3. 元注解：@Documented，保留文档注解信息
  • 不同点
    • Controller： 注册为一个控制器类
    • RestController: 等于 @Controller + @ResponseBody 注解

@ResponseBody

元注解：

  1. @Target({ElementType.TYPE, ElementType.METHOD})，作用在类和方法上
  2. @Retention(RetentionPolicy.RUNTIME)，运行期保留
  3. @Documented，保留文档注解信息

作用：

将controller的方法返回的对象通过适当的转换器转换为指定的格式之后，写入到response对象的body区，通常用来返回JSON数据或者是XML数据。

@RequestMapping

元注解：

  1. @Target({ElementType.TYPE, ElementType.METHOD})，作用在类和方法上
  2. @Retention(RetentionPolicy.RUNTIME)，运行期保留
  3. @Documented，保留文档注解信息

作用：

在Spring MVC 中使用 @RequestMapping 来映射请求，相当于配置了映射请求

@GetMapping

RequestMapping的method=”get”，作用只能在方法上

@PostMapping

RequestMapping的method=“post”，作用只能在方法上

@Autowired

@Resource

@AliasFor

注解的属性别名，如果注解放在 value 和 path 上，相当于 value 和 path 相同
